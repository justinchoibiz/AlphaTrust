# AlphaTrust

AlphaTrust is an AFML-style alpha diagnostic workspace.

It combines:

- WorldQuant-style alpha expression input
- QuantConnect-style research/backtest UX
- AFML-style diagnostics such as triple-barrier labeling, average uniqueness, purged CV, embargo, PSR, and DSR
